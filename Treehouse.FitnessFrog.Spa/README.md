
# Fitness Frog API

## Supported User Actions

* Viewing a list of entries
* Adding an entry
* Editing an entry
* Deleting an entry

## Endpoints

* /api/entries GET, POST
* /api/entries/1 GET, PUT, DELETE
* /api/activities GET
* /api/intensities GET

## Http status codes

* 200 OK
* 201 Created
* 204 No content
* 400 Bad Request
* 500 Internal Error
